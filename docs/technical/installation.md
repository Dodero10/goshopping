# Hướng dẫn cài đặt

## Cài đặt trên Windows

1. Tải xuống bộ cài đặt
2. Chạy file cài đặt
3. Làm theo hướng dẫn trên màn hình

## Cài đặt trên macOS

1. Tải xuống file .dmg
2. Mở file và kéo ứng dụng vào thư mục Applications
3. Chạy ứng dụng lần đầu
